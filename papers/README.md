# NOESIS Papers Module

`papers/` is organized as a task-first NOESIS module for paper work.

- `writing/` supports manuscript writing, section drafting, audit gates, figure/table integration, and manuscript finalization.
- `reviewing/` supports peer-review report generation, reviewer-role prompts, editor decisions, and review quality checks.
- `revising/` supports author-side revision after reviews, reviewer-response workflows, and response style policy.
- `shared/` stores reusable conventions, schemas, prompts, domain profiles, and policies used across paper activities.

Historical planning documents live in `_archive/`. Files whose classification is uncertain live in `_unsorted/`.
